# docs/founder-content-sources.md（草案）— 品牌故事來源與轉錄規則

> 本檔記錄 `/founder`（品牌故事頁）與首頁「品牌故事」區塊的**唯一授權文字來源**。
> 規則（業主 2026-07-12 明訂）：**不美化、不虛構經歷、不添加任何不存在的學歷／證照／研究背景**。
> 撰寫人稱：**第一人稱**。頁面主標為品牌故事取向（「為什麼會有科學決策」），非「創辦人」。
> 允許頭銜：創辦人／系統設計者／決策分析顧問／科學八字與科學決策模型開發者。
> 禁止頭銜：心理學家、行為科學家、科學家、心理師、研究員、命理大師、教授。

## 授權正文（業主提供正式版，逐字轉錄，不得改寫語意）

我一直很好奇一件事。
為什麼有些人明知道會後悔，卻還是一直做出同樣的選擇。

過去我接觸過不同的工作。做過吉他老師、餐飲、酒吧經營，也做過第一線服務工作。
我發現真正影響人生的，往往不是能力，而是做決定的方式。

後來我開始研究八字。但研究得越久，我越發現我真正想研究的，不是命運，而是人的決策模式。

所以我開始把八字裡真正有價值的部分拆開。拿掉吉凶、拿掉流年、拿掉神秘化的說法。
只留下能描述一個人決策傾向的結構。慢慢整理成現在的模型。這也是科學八字的開始。

而科學八字，只是整個科學決策的一部分。
科學決策不是一套算命系統。它是一套協助人理解自己、修正決策模式的分析系統。

工作分析，看的是你在工作情境下的決策傾向。
感情分析，看的是你在人際與親密關係中的決策模式。
科學八字分析，建立的是人格底色模型。
決策引擎，則協助你整理正在面對的決策。

它們的方法不同。但目的都只有一個。
不是預測未來。而是理解自己為什麼會做出現在的選擇。

我不認為任何模型可以百分之百描述一個人。所以這個系統也還在持續修正。
如果它能幫助一個人少做一次讓自己後悔的決定，那它就已經有存在的價值。
